# blog-website

## Project Title
Blog Website using HTML, CSS, and JavaScript

## Objective
To design and develop a responsive and user-friendly blog website that demonstrates core frontend development skills using HTML, CSS, and JavaScript.

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Description
This project is a simple and responsive blog website that displays multiple blog posts with titles, content, publication dates, and images.  
The website includes a navigation menu for switching between blog posts and a comment section for user interaction.  
JavaScript is used to handle comment submission dynamically without page reload.

## Features
- Multiple blog posts with images and publication dates
- Navigation menu to move between blog posts
- Comment section for each blog post
- Responsive design for mobile, tablet, and desktop
- Clean and simple user interface
- Beginner-friendly and well-structured code

## Images Used
- Images are used via direct online links (placeholder / free stock images).
- No images are downloaded or stored locally.
- Image sizing and responsiveness are handled using CSS.

## How to Use
1. Download or extract the project folder.
2. Open `index.html` in any modern web browser.
3. Navigate through blog posts using the menu.
4. Add comments by entering your name and message.
5. Comments will appear below the respective blog post.

## Code Structure
- `index.html` → Contains the structure and content of the blog website.
- `styles.css` → Handles layout, styling, responsiveness, and image sizing.
- `script.js` → Handles comment functionality using JavaScript.
